Requirement- Compare two PDF files and highlight the differences in a new result pdf file.


1. Create the Maven Project in the eclipse
2. Add the maven dependency for the pdfcomparision in POM.xml file
3. Place the 2 pdfs which supposed to be compared in the test/resources folder
4. Write the java code to compare the files.
5. Run Code the and see result.pdf file in the resource folder for the comparison results
